# Mon Portfolio - VOID_ARCHITECT_CORE

Portfolio statique moderne (Tailwind CSS, animations JS personnalisées).

## Développement local (Django)

```bash
python -m venv venv
# Windows
venv\\Scripts\\activate
# Linux/Mac
source venv/bin/activate
pip install -r requirements.txt
python manage.py migrate
python manage.py runserver
```

Visitez `http://127.0.0.1:8000/`
